# Employee-Management-System---Full-Stack-Web-Application
The Employee Management System is a web-based application developed to efficiently manage employee records within an organization. The primary objective of this project is to replace manual data handling with a digital system that enables seamless management of employee information through CRUD (Create, Read, Update, Delete) operations. The application is built using **Java, Servlet, JSP, and JDBC**, with **PostgreSQL used as the backend database.

The project follows the MVC (Model-View-Controller) architecture to ensure proper separation of concerns. The Model layer consists of the Employee class, which encapsulates employee details such as ID, name, email, and salary. The View layer is implemented using JSP pages to provide a dynamic and user-friendly interface for data input and display. The Controller layer is handled by Servlets, which process user requests, perform validation, and coordinate between the view and the data access layer. The DAO (Data Access Object) layer manages all database interactions using JDBC, ensuring efficient execution of SQL queries.

The system allows users to add new employee records, view all employees, update existing details, and delete records when necessary. PreparedStatement is used to execute parameterized queries, enhancing security by preventing SQL injection attacks. The application is deployed on an Apache Tomcat server, enabling real-time client-server interaction.

This project helped in gaining hands-on experience in Java web development, understanding HTTP request-response cycles, and implementing layered architecture. It also improved skills in database connectivity, debugging, and building scalable web applications.
